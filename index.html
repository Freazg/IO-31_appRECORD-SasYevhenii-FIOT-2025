ЛАБОРАТОРНА РОБОТА №1

Тема: ТЕГИ ТА АТРИБУТИ HTML-ДОКУМЕНТА. СТРУКТУРНА РОЗМІТКА. GIT.GITHUB. РОБОТА З РЕПОЗИТОРІЯМИ. ОПИС ЛОГІКИ ВЛАСНОГО WEB-ЗАСТОСУНКУ.

Мета роботи:

Придбати практичні навички роботи з GitHub, репозиторіями, HTML-документом, таблицями, зображеннями, посиланнями, списками, формами. Створити шаблон звітного HTML-документу для відображення результатів роботи всіх лабораторних робіт.

Опис предметного середовища:

Трекер звичок (Habit Tracker) - це веб-застосунок для формування та відстеження корисних звичок користувачів. Застосунок допомагає людям систематично виконувати необхідні дії, відстежувати прогрес і підтримувати мотивацію через візуалізацію досягнень.

Основний функціонал:

  • Управління звичками (створення, редагування, видалення)
  • Відстеження щоденного виконання
  • Візуалізація прогресу (календар, графіки, статистика)
  • Система мотивації та досягнень
  • Адаптивний дизайн для всіх пристроїв

Способи підключення стилів:

1. Зовнішній CSS файл (використовується в проєкті):

<head>
    <link rel="stylesheet" href="css/style.css">
</head>

2. Внутрішні стилі (використано в резюме):

<head>
    <style>
        body {
            background: #f4f4f4;
        }
    </style>
</head>

3. Вбудовані стилі:

<div style="color: red; padding: 10px;">Текст</div>

СЕЛЕКТОРИ

1. Селектор тегу:

HTML:

<h1>Трекер Звичок</h1>
<p>Текст параграфу</p>

CSS:

h1 {
    color: #667eea;
    font-size: 2.5em;
}

p {
    line-height: 1.6;
    color: #333;
}

Опис: Селектор тегу застосовує стилі до всіх елементів вказаного типу на сторінці.

2. Селектор класу:

HTML:

<div class="container">
    <section class="hero">Контент</section>
</div>

CSS:

.container {
    max-width: 1200px;
    margin: 0 auto;
}

.hero {
    background: white;
    padding: 40px;
    border-radius: 15px;
}

Опис: Селектор класу (починається з крапки) дозволяє застосувати стилі до елементів з певним класом. Один клас може використовуватись багато разів на сторінці.

3. Селектор ідентифікатора:

HTML:

<img id="profile-photo" src="photo.jpg" alt="Фото">

CSS:

#profile-photo {
    width: 150px;
    height: 150px;
    border-radius: 50%;
    border: 5px solid white;
}

Опис: Селектор ID (починається з #) використовується для унікальних елементів. ID має бути унікальним на всій сторінці.

4. Інші селектори:

Універсальний селектор:

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

Дочірній селектор:

header .container {
    display: flex;
    justify-content: space-between;
}

Селектор атрибута:

input[type="text"] {
    width: 100%;
    padding: 12px;
    border: 2px solid #ddd;
}

a[href] {
    color: white;
    text-decoration: none;
}

Псевдокласи:

nav a:hover {
    background: #667eea;
    color: white;
}

tbody tr:last-child td {
    border-bottom: none;
}

input:focus {
    border-color: #667eea;
}

АДАПТИВНА ВЕРСТКА - Медіа-запити

Breakpoints (точки перелому):

  • Desktop: 1200px і більше - повна версія сайту
  • Tablet: 768px - 1199px - адаптація для планшетів
  • Mobile Large: 576px - 767px - великі смартфони
  • Mobile: до 575px - маленькі смартфони

Приклади медіа-запитів з CSS:

/* Desktop - великі екрани */
@media screen and (min-width: 1200px) {
    .container {
        max-width: 1200px;
    }
}

/* Tablet - планшети */
@media screen and (max-width: 1199px) {
    .container {
        max-width: 960px;
        padding: 15px;
    }
    
    header .container {
        flex-direction: column;
        gap: 20px;
    }
}

/* Mobile - смартфони */
@media screen and (max-width: 575px) {
    body {
        font-size: 14px;
    }
    
    .hero, .features {
        padding: 15px;
    }
    
    button {
        width: 100%;
    }
}

/* Логічні оператори */
@media screen and (min-width: 768px) and (max-width: 1199px) and (orientation: landscape) {
    .hero {
        display: flex;
        align-items: center;
    }
}

ЗАВДАННЯ №3 - FLEXBOX

Варіант 10 - Навчальний центр IT

Створено повноцінний адаптивний сайт навчального центру з використанням Flexbox.

Основні Flexbox властивості:

/* Flex Container */
.header-content {
    display: flex;
    justify-content: space-between;
    align-items: center;
    gap: 20px;
}

/* Flex Items */
.nav {
    display: flex;
    gap: 30px;
}

/* Responsive Flexbox */
.services-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 30px;
    justify-content: center;
}

.service-card {
    width: calc(33.333% - 20px);
    min-width: 280px;
}

@media (max-width: 768px) {
    .header-content {
        flex-direction: column;
    }
    
    .service-card {
        width: 100%;
    }
}

Переваги Flexbox:

  • Легке вирівнювання елементів по горизонталі та вертикалі
  • Адаптивність без складних обчислень
  • Зручне управління пробілами між елементами (gap)
  • Можливість змінювати порядок елементів
  • Автоматичний розподіл простору
