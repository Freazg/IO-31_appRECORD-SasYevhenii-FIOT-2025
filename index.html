Мета роботи:

Придбати практичні навички роботи з GitHub, репозиторіями, HTML-документом, таблицями, зображеннями, посиланнями, списками, формами. Створити шаблон звітного HTML-документу для відображення результатів роботи всіх лабораторних робіт.

Опис предметного середовища:

Трекер звичок (Habit Tracker) - це веб-застосунок для формування та відстеження корисних звичок користувачів. Застосунок допомагає людям систематично виконувати необхідні дії, відстежувати прогрес і підтримувати мотивацію через візуалізацію досягнень.

Основний функціонал:

  • Управління звичками (створення, редагування, видалення)
  • Відстеження щоденного виконання
  • Візуалізація прогресу (календар, графіки, статистика)
  • Система мотивації та досягнень
  • Адаптивний дизайн для всіх пристроїв

Структура головної сторінки WEB-застосунку:

Головна сторінка містить наступні HTML-елементи:

1. Таблиця - Порівняння функціоналу

<table>
    <thead>
        <tr>
            <th>Функція</th>
            <th>Безкоштовна версія</th>
            <th>Преміум версія</th>
            <th>Опис</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Кількість звичок</td>
            <td>До 5</td>
            <td>Необмежено</td>
            <td>Відстежуйте всі необхідні звички</td>
        </tr>
    </tbody>
</table>

Опис тегів таблиці:

  • <table> - основний контейнер таблиці
  • <thead> - заголовок таблиці
  • <tbody> - тіло таблиці з даними
  • <tr> - рядок таблиці
  • <th> - комірка заголовка (жирний текст)
  • <td> - звичайна комірка даних

2. Зображення

<img src="https://images.unsplash.com/photo-1484480974693-6ca0a78fb36b?w=600" 
     alt="Трекер звичок" 
     class="hero-image">

Атрибути тегу img:

  • src - шлях до зображення (URL або локальний файл)
  • alt - альтернативний текст (для доступності та SEO)
  • width - ширина зображення
  • height - висота зображення
  • class - CSS клас для стилізації

3. Списки

<ol>
    <li><strong>Пити 2 літри води щодня</strong>
        <ul class="nested-list">
            <li>Покращує метаболізм</li>
            <li>Підтримує здоров'я шкіри</li>
        </ul>
    </li>
    <li><strong>Ранкова зарядка</strong></li>
</ol>

Типи списків:

  • <ul> - ненумерований (маркірований) список
  • <ol> - нумерований список
  • <li> - елемент списку

Атрибути:

  • type - тип маркера або нумерації (для ol: 1, A, a, I, i)
  • start - початкове значення нумерації

4. Форма створення звички

<form>
    <div class="form-group">
        <label for="habitName">Назва звички*</label>
        <input type="text" id="habitName" required>
    </div>
    
    <div class="form-group">
        <label for="habitCategory">Категорія*</label>
        <select id="habitCategory" required>
            <option value="">Обери категорію</option>
            <option value="health">Здоров'я та фітнес</option>
        </select>
    </div>
    
    <div class="form-group">
        <label>Частота виконання*</label>
        <input type="radio" name="frequency" value="daily" checked> Щодня
        <input type="radio" name="frequency" value="weekly"> Тижнево
    </div>
    
    <button type="submit">Створити звичку</button>
</form>

Теги форми:

  • <form> - контейнер форми
  • <input> - поле введення різних типів
  • <label> - підпис для поля
  • <select> - випадаючий список
  • <option> - опція у випадаючому списку
  • <textarea> - багаторядкове текстове поле
  • <button> - кнопка
