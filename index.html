ЛАБОРАТОРНА РОБОТА №1

Тема: ТЕГИ ТА АТРИБУТИ HTML-ДОКУМЕНТА. СТРУКТУРНА РОЗМІТКА. GIT.GITHUB. РОБОТА З РЕПОЗИТОРІЯМИ. ОПИС ЛОГІКИ ВЛАСНОГО WEB-ЗАСТОСУНКУ.

Мета роботи:

Придбати практичні навички роботи з GitHub, репозиторіями, HTML-документом, таблицями, зображеннями, посиланнями, списками, формами. Створити шаблон звітного HTML-документу для відображення результатів роботи всіх лабораторних робіт.

Опис предметного середовища:

Трекер звичок (Habit Tracker) - це веб-застосунок для формування та відстеження корисних звичок користувачів. Застосунок допомагає людям систематично виконувати необхідні дії, відстежувати прогрес і підтримувати мотивацію через візуалізацію досягнень.

Основний функціонал:

  • Управління звичками (створення, редагування, видалення)
  • Відстеження щоденного виконання
  • Візуалізація прогресу (календар, графіки, статистика)
  • Система мотивації та досягнень
  • Адаптивний дизайн для всіх пристроїв

ЛАБОРАТОРНА РОБОТА №2

Тема: КАСКАДНІ ТАБЛИЦІ СТИЛІВ. СЕЛЕКТОРИ ТЕГУ, КЛАСУ, ІДЕНТИФІКАТОРА. СТИЛЬОВЕ ОФОРМЛЕННЯ ТЕКСТОВИХ ЕЛЕМЕНТІВ В HTML-ДОКУМЕНТАХ.

Мета роботи:

Придбати практичні навички роботи з селекторами, ідентифікаторами, списками, різноманітними властивостями кольору і фону, зовнішними та внутрішніми відступами, плаваючими елементами, оформленням текстових елементів.

Посилання на ресурси:

Назва Посилання
WEB-застосунок з CSS Live Page
Резюме Resume Page
CSS файл style.css

Способи підключення стилів:

1. Зовнішній CSS файл (використовується в проєкті):

<head>
    <link rel="stylesheet" href="css/style.css">
</head>

2. Внутрішні стилі (використано в резюме):

<head>
    <style>
        body {
            background: #f4f4f4;
        }
    </style>
</head>

3. Вбудовані стилі:

<div style="color: red; padding: 10px;">Текст</div>

СЕЛЕКТОРИ

1. Селектор тегу:

HTML:

<h1>Трекер Звичок</h1>
<p>Текст параграфу</p>

CSS:

h1 {
    color: #667eea;
    font-size: 2.5em;
}

p {
    line-height: 1.6;
    color: #333;
}

Опис: Селектор тегу застосовує стилі до всіх елементів вказаного типу на сторінці.

2. Селектор класу:

HTML:

<div class="container">
    <section class="hero">Контент</section>
</div>

CSS:

.container {
    max-width: 1200px;
    margin: 0 auto;
}

.hero {
    background: white;
    padding: 40px;
    border-radius: 15px;
}

Опис: Селектор класу (починається з крапки) дозволяє застосувати стилі до елементів з певним класом. Один клас може використовуватись багато разів на сторінці.

3. Селектор ідентифікатора:

HTML:

<img id="profile-photo" src="photo.jpg" alt="Фото">

CSS:

#profile-photo {
    width: 150px;
    height: 150px;
    border-radius: 50%;
    border: 5px solid white;
}

Опис: Селектор ID (починається з #) використовується для унікальних елементів. ID має бути унікальним на всій сторінці.

4. Інші селектори:

Універсальний селектор:

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

Дочірній селектор:

header .container {
    display: flex;
    justify-content: space-between;
}

Селектор атрибута:

input[type="text"] {
    width: 100%;
    padding: 12px;
    border: 2px solid #ddd;
}

a[href] {
    color: white;
    text-decoration: none;
}

Псевдокласи:

nav a:hover {
    background: #667eea;
    color: white;
}

tbody tr:last-child td {
    border-bottom: none;
}

input:focus {
    border-color: #667eea;
}

CSS ВЛАСТИВОСТІ

Шрифти:

body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    font-size: 16px;
    line-height: 1.6;
}

h1 {
    font-weight: bold;
    font-size: 2.5em;
}

Колір тексту та фон:

body {
    color: #333;
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

.hero {
    background: white;
    color: #333;
}

Контури (borders):

    border: 2px solid #ddd;
    border-radius: 8px;
}

.section-title {
    border-left: 5px solid #667eea;
    padding-left: 15px;
}

Відступи (padding та margin):

    padding: 20px;
    margin: 0 auto;
}

.hero {
    padding: 40px;
    margin-bottom: 30px;
}

Таблиці:

    width: 100%;
    border-collapse: collapse;
}

    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
}

th, td {
    padding: 15px;
    text-align: left;
    border-bottom: 1px solid #ddd;
}

Списки:

ul, ol {
    margin-left: 30px;
    margin-bottom: 20px;
}

nav ul {
    list-style: none;
    display: flex;
    gap: 20px;
}

CSS Просунутий (flexbox, transitions, shadows):

/* Flexbox */
.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

/* Transitions */
button {
    transition: transform 0.2s, box-shadow 0.2s;
}

button:hover {
    transform: translateY(-2px);
    box-shadow: 0 5px 15px rgba(102, 126, 234, 0.4);
}

/* Box Shadow */
.hero {
    box-shadow: 0 10px 30px rgba(0,0,0,0.2);
}

/* Border Radius */
.card {
    border-radius: 15px;
}
